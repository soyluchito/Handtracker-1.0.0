Bienvenido a la documentación de Handtracker 1.0.0

Este es un proyecto de detección de manos en tiempo real usando Python, OpenCV y MediaPipe.

Es una aplicación que detecta y sigue manos en tiempo real utilizando visión por computadora. El sistema identifica landmarks de la mano y permite futuras implementaciones como control por gestos o interacción con sonidos.

TECNOLOGÍAS USADAS:
- Python
- OpenCV
- MediaPipe
- Pygame

Uso:
```bash
pip install -r requirements.txt
python main.py
